# Book Tracker Rewrite in Java Spring

## Purpose:
This project is a rewrite/refactor of the Book Tracker Project I completed for my AAS Computer Programming and Software Development Capstone Project. With this project, I will be able to write the program in Java with Spring (rather than Python with Flask). Additionally, I will be able to incorporate stronger OOP principles with this rewrite.  

Link to Original Project: [https://github.com/collingeersen/Book-Tracker](https://github.com/collingeersen/Book-Tracker)

## List of Differences from the Original Project:
|                   | Original                  | Rewrite |
|-------------------|---------------------------|---------|
| Language          | Python                    | Java    |
| Framework         | Flask                     | Spring  |
| Database          | SQLite                    | Oracle  |
| Testing Framework | pytest                    | JUnit   |
| Release Format    | .whl and Docker container | .jar    |
| Language |  |  |
